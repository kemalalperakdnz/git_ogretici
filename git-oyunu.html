Git Komut Rehberi

Temel Komutlar

  • git init - Yeni bir Git deposu oluşturur
  • git status - Çalışma dizininin durumunu gösterir
  • git add <dosya> - Dosyayı hazırlama alanına ekler
  • git commit -m "mesaj" - Değişiklikleri kaydeder

Dal Komutları

  • git branch - Mevcut dalları listeler
  • git branch <dal-adı> - Yeni bir dal oluşturur
  • git checkout <dal-adı> - Belirtilen dala geçer
  • git merge <dal-adı> - Belirtilen dalı mevcut dala birleştirir

Uzak Repo Komutları

  • git remote add <ad> <url> - Uzak repo ekler
  • git push <uzak> <dal> - Değişiklikleri uzak repoya gönderir
  • git pull <uzak> <dal> - Değişiklikleri uzak repodan çeker
  • git clone <url> - Uzak repoyu klonlar
